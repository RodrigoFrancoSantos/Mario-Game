🍄 Mario Jump Game 🎮
Um mini-jogo do Mario feito com HTML, CSS e JavaScript puro — desenvolvido como projeto de estudos de Front-End. 🚀

Mario

📖 Sobre o projeto
Este projeto foi criado com o objetivo de praticar e aprofundar conhecimentos em CSS e JavaScript, recriando uma versão simplificada do clássico jogo do Super Mario Bros. 🌟

A ideia principal foi explorar:

🎨 CSS animations (@keyframes, animation, transition)
🧩 Posicionamento com position: absolute e relative
🕹️ Manipulação do DOM com JavaScript
⌨️ Eventos de teclado para fazer o Mario pular
💥 Detecção de colisão simples entre o Mario e o cano
🛠️ Tecnologias utilizadas
Tecnologia	Para quê
🌐 HTML5	Estrutura da página e do cenário
🎨 CSS3	Estilos, cenário, animações e movimento do cano/nuvens
⚡ JavaScript (Vanilla)	Lógica de pulo, colisão e game over

Sem frameworks, sem bibliotecas. Só o básico bem feito. 💪


    
✨ Funcionalidades

✅ Cenário animado com nuvens em movimento ☁️
✅ Cano que se move da direita para a esquerda 🟢
✅ Personagem Mario com animação de corrida 🏃
✅ Ação de pular ao pressionar uma tecla 🆙
✅ Detecção de colisão entre Mario e cano 💥
✅ Tela de Game Over quando o Mario é atingido 💀

🧠 O que aprendi

🔍 Como usar setInterval para criar loops de jogo
🎞️ Como construir animações fluidas usando apenas CSS
📐 Como capturar a posição de elementos com getComputedStyle e offsetLeft
🧱 Como pensar em lógica de jogo em pequena escala
🧹 Como manter o código organizado e separado (HTML / CSS / JS)
